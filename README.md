# Compute Freely

Compute Freely is a directory and promise page for a suite of free, local-first browser tools.

Live site: https://computefreely.com/

## Tools

- Record Freely: https://recordfreely.com/
- QR Freely: https://qrfreely.com/
- Video Freely: https://videofreely.com/
- PDF Freely: https://pdffreely.com/
- Fill Freely: https://fillfreely.com/
- Image Freely: https://imagefreely.com/
- Resume Freely: https://resumefreely.com/

## Promise

Freely tools are built to avoid the common traps in online utilities:

- no surprise paywalls
- no account requirement
- no watermark
- no download gate
- no hidden upload
- no bait-and-switch redirects

Where practical, tools run locally in the browser and keep files on the user's device.

## Run Locally

This is a static site. From this directory:

```sh
python3 -m http.server 4172
```

Then open `http://localhost:4172`.
